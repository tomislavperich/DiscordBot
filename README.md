# DiscordBot

## Prerequisites
1. Python 3+
2. [Discord API wrapper](https://github.com/Rapptz/discord.py)

    or use
```bash
python pip -m install discord.py
```

## Usage
1. Acquire an access token
      - Create an account on Discord
      - Create an app on Discord
      - Convert app to user
      - Copy the token to code

2. Run DiscordBot.py
